# 나의 할 일 관리 앱

## 1. 앱 소개

이 앱은 사용자가 할 일을 입력하고 관리할 수 있는 간단한 Todo List 앱입니다.

## 2. 사용한 AI

- ChatGPT

## 3. 처음 입력한 프롬프트

HTML, CSS, JavaScript를 하나의 index.html 파일에 넣어서 할 일 관리 앱을 만들어줘. 할 일을 추가하고 완료 표시와 삭제 기능이 있었으면 좋겠어.

## 4. 수정 요청한 내용

수정 요청: 완료 표시 가능 코드가 있는지 알려주고 없으면 만들어줘

## 5. 어려웠던 점 또는 배운 점

JavaScript를 이용해 버튼 클릭 시 할 일을 추가하고 삭제하는 기능을 구현하는 방법을 배웠다. 또한 CSS를 사용해 화면 디자인을 꾸미는 방법을 배웠다.
